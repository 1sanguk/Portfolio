# 박상욱 포트폴리오

서버/백엔드 개발자 박상욱(Steve Park)의 이력 및 프로젝트를 소개하는 정적 웹사이트입니다.

🔗 https://1sanguk.github.io/Portfolio/

## 구성

- `index.html` — 학력, 경력, 어학/자격증, 활용능력, 대표 프로젝트(STAR), 프로젝트 목록을 담은 단일 페이지
- `style.css` — 전체 스타일 (rem 기반 반응형 타이포그래피)
- `script.js` — 섹션 펼치기/접기, 헤더 미니 목차 스크롤스파이, 프로젝트/대표 프로젝트 카드 펼침 인터랙션
- `images/` — 프로젝트별 스크린샷

## 주요 기능

- 섹션별 접기/펼치기 (학력사항, 경력사항, 활용능력, 대표 프로젝트, 프로젝트 목록 등)
- 헤더의 미니 목차가 현재 보고 있는 섹션에 맞춰 자동으로 활성화
- 대표 프로젝트 2~3개를 STAR(상황-과제-행동-결과) 구조로 정리, 카드별로 펼치기/접기 가능
- 프로젝트 목록에서 카드를 클릭하면 장르, 상세 설명, 스크린샷이 펼쳐짐
- 회사·기간별로 그룹화된 프로젝트 타임라인
