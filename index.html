  • Claude API 연동 및 질문 자동 분류 라우팅, FastAPI + SSE 스트리밍 서빙→ 동일한 RAG 검색기를 재사용하고 생성 노드만 Claude API(claude-haiku-4-5)로 교체해 SOP_GPT와 실시간 비교하는 분할화면 UI를 구성했습니다. Claude Haiku(temperature=0)로 질문을 잡담/사실형/일반으로 자동 분류해 적합한 체인으로 라우팅하며, FastAPI + SSE 스트리밍으로 서빙합니다.
